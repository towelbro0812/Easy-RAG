## 開啟方式
1. 創建conda環境並且進入環境
   ```bash
   conda create -n RAG python==3.10
   conda activate RAG
   ```
2. 安裝依賴庫
   ```bash
   pip install -r requirements.txt
   ```
3. 開啟`mian.ipynb`
